## EIN Usage
确保在 `python-mode` 模式下：
- `ein:jupyter-server-start` 启动之后在浏览器访问 http://localhost:8888/tree ，就可以看到 Jupyter 页面
- `ein:jupyter-server-stop`
- `ein:notebooklist-login` 登陆授权，以便后续操作
- `ein:notebook-open`
